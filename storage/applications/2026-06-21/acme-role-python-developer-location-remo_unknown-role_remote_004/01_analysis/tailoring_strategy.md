# Tailoring Strategy

- Main selling points: emphasize approved experience from the master CV.
- Skills to emphasize: approved experience from the master CV.
- Projects to emphasize: relevant existing projects from the CV.
- Experience bullets to prioritize: backend, reliability, data, and user-facing impact where supported.
- Keywords to include only if supported: Django, Go, Python, SQL.
- Things to avoid: unsupported years of experience, invented employers, invented seniority.
- Missing skills to handle carefully: Python, Django, SQL, Go.
- Cover letter angle: connect profile strengths to the role.
- Salary guidance notes: use only manually approved salary expectations.
